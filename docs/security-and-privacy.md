# Безопасность и приватность

Этот репозиторий — публичное описание реального коммерческого проекта Pegas Auto CRM.

Полный коммерческий код, закрытые настройки, ключи, данные клиентов и непубличные материалы заказчика не публикуются.

В публичную версию включены только:

- краткое описание проекта;
- архитектурный обзор без приватных деталей;
- скриншоты интерфейса с демонстрационными или обезличенными данными;
- безопасные TypeScript-примеры отдельных модулей.

Цель репозитория — показать бизнес-задачу, интерфейс, архитектурный подход и инженерный вклад без раскрытия коммерчески чувствительной информации.
