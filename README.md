# Pegas Auto CRM — CRM/POS-система для сети шиномонтажей

Реальный коммерческий проект: веб-система для сети шиномонтажей.  
Система помогает вести заказы, клиентов, автомобили, запись, смены, услуги, зарплаты, аналитику и онлайн-запись в одном интерфейсе.

## Задача

Нужно было заменить разрозненные таблицы и ручные процессы на единую систему под работу шиномонтажа: несколько филиалов, посты записи, касса, сотрудники, услуги, зарплаты и история клиентов.

## Что реализовано

- авторизация сотрудников;
- доступ к данным по филиалам;
- открытие и закрытие смен;
- касса и список заказов смены;
- создание и ведение заказов;
- карточка заказа с клиентом, автомобилем, услугами и оплатой;
- база клиентов и автомобилей;
- календарь записи по постам;
- публичная онлайн-запись для клиентов;
- настройка услуг, категорий и прайсов;
- расчёт зарплаты исполнителей;
- роли и права доступа;
- базовая аналитика по заказам, выручке, услугам и выплатам;
- печать квитанции 57 мм без фискализации.

## Архитектура

- **Frontend:** TypeScript, Next.js, рабочий веб-интерфейс для сотрудников, администраторов и публичной онлайн-записи.
- **Backend/runtime:** серверная логика заказов, смен, записи, ролей, кассы и зарплат.
- **Database:** PostgreSQL для хранения клиентов, автомобилей, заказов, услуг, смен, оплат, записей и прав доступа.
- **ORM:** Prisma.
- **Access control:** роли сотрудников, администраторов и владельца.
- **Branch-aware access:** доступ к данным ограничивается филиалами и рабочими зонами сотрудника.
- **Business flow:** клиент → автомобиль → заказ → услуги → исполнитель → оплата → смена → зарплата → аналитика.

## Мой вклад

Проект выполнен как solo/full-stack разработка: архитектура, frontend, backend, база данных, бизнес-логика, роли, поддержка и дальнейшие доработки.

## Что не публикуется

Полный коммерческий код, настройки, ключи, данные клиентов и закрытые материалы заказчика не публикуются.  
В репозитории собраны публичное описание проекта, скриншоты интерфейса и безопасные примеры отдельных модулей.
